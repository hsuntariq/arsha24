Learn the basics of Python programming with hands-on examples.

Python for Beginners Tutorial

Master variables, loops, and functions with this beginner-friendly guide to Python programming.

1. Introduction to Python

Python is a versatile, high-level programming language known for its simplicity and readability. It’s widely used in web development, data science, and automation. This tutorial covers the basics to get you started.

Installing Python

Download Python from python.org and install it on your system. Verify the installation by running the following command in your terminal: 

python --version

Variables in Python

Variables store data. In Python, you don’t need to declare a variable’s type. Here’s an example: 

name = "Alice"
age = 25
height = 5.6
print(name, age, height)

This code assigns a string, integer, and float to variables and prints them.

Loops in Python

Loops allow repetitive tasks. The for loop iterates over a sequence: 

for i in range(5):
    print(i)

This prints numbers 0 to 4. The while loop continues until a condition is false: 

count = 0
while count < 5:
    print(count)
    count += 1

Functions in Python

Functions encapsulate reusable code. Define a function using the def keyword: 

def greet(name):
    return f"Hello, {name}!"
print(greet("Bob"))

This function takes a name parameter and returns a greeting.

Practical Example: Simple Calculator

Let’s build a basic calculator to practice these concepts:

def calculator():
    num1 = float(input("Enter first number: "))
    op = input("Enter operator (+, -, *, /): ")
    num2 = float(input("Enter second number: "))
    
    if op == "+":
        result = num1 + num2
    elif op == "-":
        result = num1 - num2
    elif op == "*":
        result = num1 * num2
    elif op == "/":
        result = num1 / num2 if num2 != 0 else "Error: Division by zero"
    else:
        result = "Invalid operator"
    
    print(f"Result: {result}")

calculator()

This program prompts for two numbers and an operator, performs the calculation, and displays the result.

Next Steps

Practice by modifying the calculator to handle more operators or add error handling. Explore Python’s libraries like math or random for advanced functionality.
