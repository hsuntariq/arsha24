Chapter 5: Python Variables

Learn how to create and use variables to store data in Python programs.

Understanding Python Variables

Variables in Python are used to store data that can be referenced and manipulated in a program. Think of a variable as a labeled container that holds a value, such as a number, string, or other data type. Python’s dynamic typing means you don’t need to declare a variable’s type explicitly; the interpreter assigns it based on the value. This chapter covers creating variables, naming rules, and basic operations.

Creating Variables

To create a variable, assign a value to a name using the equals sign (=): 

x = 10  # Integer variable
name = "Alice"  # String variable
price = 19.99  # Float variable
Here, x holds an integer, name holds a string, and price holds a float. You can print variables using the print() function: 
print(x)  # Outputs: 10
print(name)  # Outputs: Alice

Variable Naming Rules

Python has specific rules for naming variables:

  • Names must start with a letter (a-z, A-Z) or underscore (_).
  • Names can contain letters, numbers (0-9), and underscores.
  • Names are case-sensitive (age and Age are different).
  • Names cannot be Python keywords (e.g., if, for, while).
Examples of valid and invalid names: 
# Valid
my_variable = 5
user_name = "Bob"
_age = 25

# Invalid
2var = 10  # Starts with a number
my-var = 20  # Contains hyphen
for = 30  # Uses keyword
Follow Python’s PEP 8 style guide, which recommends using lowercase with underscores for variable names (e.g., user_age).

Assigning Multiple Variables

You can assign multiple variables in one line: 

a, b, c = 1, 2, 3
print(a, b, c)  # Outputs: 1 2 3
Or assign the same value to multiple variables: 
x = y = z = 10
print(x, y, z)  # Outputs: 10 10 10

Variable Types

Python variables can hold different data types (covered in detail in Chapter 6). Common types include:

  • int: Integer (e.g., 10)
  • float: Floating-point number (e.g., 3.14)
  • str: String (e.g., "Hello")
  • bool: Boolean (e.g., True, False)
You can check a variable’s type using type(): 
x = 42
print(type(x))  # Outputs:
